<ol>
    <li>Install global dependencies: </li>
    <pre>npm install -g prisma graphql tsc</pre>
    <li>Install logdependencies: </li>
    <pre>npm install</pre>
    <li>Configure Variables: </li>
    <pre>nano ./variables.env</pre>
    <li>Run Server: </li>
    <pre>tsc ./src/index.ts && npm run debug</pre>
